# Aleator
Current Game Project (Probably)

Aleotor 게임 설명

Aleotor는 야추(Yahtzee)의 족보 규칙과 발라트로(Balatro) 스타일의 점수 연출을 융합한 로그라이트 주사위 게임입니다.
플레이어는 제한된 주사위를 굴려 족보를 완성하고, 이를 통해 **칩(Chips)**과 **배수(Multiplier)**를 얻어 점수를 쌓아갑니다.

게임의 핵심은 단순한 굴림 → 족보 완성 → 점수 계산의 흐름을 확장해,
아이템과 시너지를 통해 점점 더 큰 수치를 만들어내는 쾌감을 주는 것입니다.

🎲 주요 특징

야추 규칙 기반: 족보(예: 포카드, 풀하우스, 스트레이트 등)를 채워 점수를 획득

칩 × 배수 시스템: 칩 수치와 배수가 곱해져 최종 점수가 결정됨

발라트로 스타일 타이밍: 점수가 실시간으로 올라가는 애니메이션, 단계별 시각 효과

아이템 시스템

테세라(Tessera): 특정 조건에서 추가 칩/배수 효과 발동

인장(Sigillum): 족보 보너스를 강화

점성술(Astrologia): 즉시 발동하는 원샷 효과

공인서(Diploma): 런 전체 규칙을 바꾸는 지속 효과

조약(Pactum): 강력한 보상과 디버프를 동시에 주는 계약

⚡ 플레이 흐름

주사위 굴림 → 원하는 조합 선택

족보에 배치 → 기본 칩/배수 획득

아이템 발동 → 조건에 따라 추가 효과

점수 계산 → 칩 × 배수로 최종 점수 확정

다음 라운드 진행 → 점차 높은 점수와 시너지 추구

🎯 목표

제한된 라운드 안에 목표 점수를 돌파

더 높은 난이도, 변칙 룰, 강력한 아이템 조합을 통해 폭발적인 점수 성장을 경험
